Projects

Ketto

Developed a website clone mimicking the functionalities of the real Ketto fundraising organization website. Designed full website responsive, can adopt every device like phone, laptop, tablet. Completed as an individual project within a timeframe of 5 days.

omdb search

The OMDB search project utilizes the IMDb API to allow users to search for movies, TV shows, and other media content. It implements a debounce function to optimize the search functionality by delaying the API call until the user stops typing, thereby reducing unnecessary API requests and improving performance.

Food Ordering System

This system designed to streamline the process of placing and processing orders asynchronously. Users can select items from a menu, place their order, and then observe the status of their order as it progresses. Once an order is processed, the system displays the ordered items along with their images and positions them at the top for easy visibility.

Basic To-Do List App

To-Do list app allows users to add tasks and assign priority levels to them. Users can input tasks they need to complete, designate whether each task is of high or low priority, and manage their to-do list accordingly. This app helps users organize their tasks more effectively.

Basic E-Commerce Website Fuctionality

This basic e-commerce website consisting of three main pages: home, cart, and checkout. Users can browse items on the home page, add them to their cart, and view the total number of items in the cart. In the cart page, users can remove items, view the total amount of their cart, and proceed to checkout. At the checkout page, users can input their name to place the order. This project serves as a fundamental introduction to e-commerce website development, focusing on features like product browsing, cart management, and checkout process.

Amazon Clone

The Amazon clone project is a beginner-level endeavor aimed at replicating the basic functionalities of the Amazon e-commerce platform. It involves creating a simplified version of Amazon's interface, allowing users to browse products.
